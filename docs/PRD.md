# Project Vibe: Solana Address Sniper & Monitor

## 1. 项目目标
建立一个自用的 Solana 地址实时监控系统。当目标地址发生交易（Swap/Transfer）时，立即通过 Telegram 推送详细分析。

## 2. 核心功能
* **实时监控：** 监听指定 Solana 地址列表的所有链上活动。
* **智能分析：** 识别交易类型（买入/卖出/转账）、Token 名称、金额及所属平台（Raydium/Jupiter）。
* **TG 提醒：** 推送格式化后的消息，包含 Solscan 链接和交易摘要。
* **管理后台：** 一个简单的 Web 界面，用于添加/删除监控地址，查看最近 50 条监控记录。

## 3. 用户故事 (User Persona)
"作为一个链上猎手，我希望在聪明钱进场后的 5 秒内收到通知，并能在手机上通过网页快速调整我的监控名单。"
